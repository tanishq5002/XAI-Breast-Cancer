About the Project

Explainable Deep Learning framework for breast cancer detection using histopathology images.

Objective

The goal of this project is to assist medical professionals by providing an automated decision-support system capable of identifying cancerous patterns in histopathology images. The system enhances trust by offering Grad-CAM based explainability to visualize regions influencing predictions.

Key Features

• Upload histopathology images for analysis
• Automatic benign vs malignant classification
• Confidence score visualization
• Grad-CAM explainability heatmaps
• Interactive and user-friendly interface

Technology Stack

Python Flask TensorFlow Keras OpenCV Grad-CAM HTML CSS

Dataset

The model is trained on histopathology breast cancer datasets consisting of microscopic tissue images. These images are preprocessed, normalized, and used to train a CNN model for binary classification.
